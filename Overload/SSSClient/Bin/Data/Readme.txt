1. 반드시 SSSScene, SSSPrefab만 넣도록 한다.
2. SSSScene 파일은 리소스 데이터를 포함하지 않고, 파일을 생성 할 당시의 리소스 파일의 디렉토리를 저장한다. 
따라서 저장할 당시의 리소스 디렉토리와 일치하지 않으면 예상하지 않은 결과가 나올 수 있다.

3. SSSPrefab 파일은 마찬가지로 리소스 데이터를 포함하지 않고 파일을 생성할 당시의 리소스 파일의 디렉토리를 저장한다.
SSSScene의 경우와 같이 저장할 당시의 리소스파일의 디렉토리가 일치하지 않으면 예상하지 않은 결과가 나올 수 있다.

4. 폴더에 분류될 사항은 다음과 같다.
- 폴더 명에는 띄어쓰기를 하지 않도록 한다.(언더스코어 "_"로 연결)
- SSSScene 파일은 Data\Scene에 저장하도록 한다. 
- SSSPrefab 파일은 Data\Prefab에 저장하도록 한다.
- 경우에 따라 하위에 Level별로 분류하도록 한다. 필요하지 않다면 굳이 하지 않아도 된다.
